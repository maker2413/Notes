Assembly language, assembly, and commonly abbreviated as ASM or asm, is any
low-level programming language with a very strong correspondence between the
instructions in the language and the architecture's machine code instructions.

Assembly is consider a second generation [[id:94903e09-f03d-4b20-b2eb-1da7618282ee][programming language]].
