Org Roam is a package for [[id:d18ed6e8-dbce-4822-9d3f-3de38246ba38][Emacs]] that lets you take [[id:3336e7d9-ec1f-4d21-be4b-ae4a7cba7c8f][Org Mode]] based notes. It is
intended for linking concepts together using the [[id:06bff409-7879-43e6-844c-70464924c86b][Zettelkasten]] note taking
method.
